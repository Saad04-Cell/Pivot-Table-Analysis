# Excel Project – Pivot Table Analysis
Built Pivot Tables to analyze sales data and answer business questions. The project summarizes order quantities, compares sales by order priority, filters yearly sales, identifies the least profitable region, and determines the customer segment with the highest sales using Excel Pivot Tables.

## 📌 Project Overview

This project demonstrates how Microsoft Excel Pivot Tables can be used to summarize sales data and answer real business questions. The analysis provides insights into sales performance, customer behavior, and regional profitability to support data-driven decision-making.

---

## 🎯 Business Questions Answered

- What is the total order quantity for Liz Pelletier?
- Which Order Priority generated the highest sales?
- What were the total sales made in 2009?
- Which region recorded the least profit in 2012?
- Which customer segment generated the highest sales?
